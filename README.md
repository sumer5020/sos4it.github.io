# Single page project

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Compiles css for development
```
npm run dev
```

### Compiles and minifies css for production
```
npm run prod
```
